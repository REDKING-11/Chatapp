**DM Security Test Suite**
- Added dedicated Node-run security tests for the DM crypto path.
- Covered key wrapping and unwrap failures when wrapped key material, IVs, auth tags, or ephemeral public keys are tampered with.
- Covered encrypted payload round-trips and negative paths for modified ciphertext, nonce, tag, and AAD.
- Covered device bundle signature checks, message envelope signature checks, and device-transfer payload tampering.
- Added service-level tests for encrypted message receive behavior, replay detection, duplicate decrypted message ids, signature failures, decryption failures, wrong-recipient wrapped keys, and revoked or invalid sender device material.
- Added storage-boundary tests for encrypted-at-rest DM store reads/writes, corrupted encrypted payload quarantine, corrupted master key quarantine, final-file writes, and legacy source migration.
- Added narrow internal test hooks so the DM storage and service modules can be tested under plain Node without needing Electron at runtime.
