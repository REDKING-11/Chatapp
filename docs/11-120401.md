# LibreChat — Change Log (11-120401)

Worklog / Ongoing Improvements Alpha 0.1.3

---

## Core Text & Content

* Added `Pretext.js` support for text-like builder blocks:

  * Text and heading nodes now use an arithmetic layout engine instead of relying on browser DOM measurement
* Added broad Markdown support across:

  * Builder content
  * Chat messages
  * DMs and related text surfaces
* Added live Markdown preview while typing
* Expanded Markdown support to include:

  * Headings, emphasis, code blocks
  * Blockquotes, lists, task lists
  * Tables, horizontal rules
  * Links and images
* Added:

  * Mermaid diagram rendering
  * Simple chart rendering from Markdown code fences
* Markdown rendering remains sanitized (no raw HTML/script execution)

---

## Reactions & Message UX

* Added Discord-style reactions:

  * Quick emoji options
  * Full emoji picker
* Reworked reaction system:

  * Reactions attach to messages instead of appearing as standalone entries
  * Reaction chips render under the target message
* Reworked message footer:

  * Message actions and reaction controls now live under the message
  * Prevents message bubble stretching

### Fixes

* Fixed reaction picker issues:

  * Sticky open state
  * Outside-click close
  * Picker centering
  * Backdrop styling
  * Multi-click behavior
* Fixed DM reaction pipeline:

  * Reactions rebuild into target messages after reload
  * Removed “ghost reaction messages”
* Added and refined overlay-based reaction picker:

  * Fixed emoji selection wiring
  * Tightened open/close logic after regressions

---

## Message Composer

* Added emoji insertion into the composer (separate from reactions)
* Added basic file sending support
* Redesigned composer layout:

  * File button on the left
  * Emoji trigger inside the input field
  * Smaller send button on the right
* Made text area non-resizable to prevent layout stretching
* Restored animated “dancing emoji” trigger:

  * Rotates emoji every 20 seconds

### Fixes

* Fixed emoji trigger centering (aligned with paperclip logic)
* Fixed emoji picker backdrop covering the entire screen
* Updated emoji picker behavior:

  * Stays open after selection for easier multi-insert use

---

## File Sharing & Attachments

* Initially added lightweight file sharing via Markdown/data URLs
* Replaced with secure no-storage attachment flow for DMs/groups:

  * Sender sends attachment metadata instead of full file blob
  * Recipient receives a download button
  * File transfer streams live
  * Both sides show progress
  * No long-term server storage required

### Improvements

* Polished attachment cards:

  * File metadata display
  * Status indicators
  * Progress UI
  * Cleaner layout and alignment
* Improved long filename handling
* Centered remove (`x`) button

### Notes

* Current attachment system is suited for smaller files
* Larger files (e.g. modpacks) will require hosted or P2P solutions later

---

## DM & Group Chat Systems

* Added disappearing messages:

  * Separate from relay retention
* Added agreement flow:

  * Both participants must accept policy changes
* Added additional duration options:

  * `2 months`, `4 months`, `6 months`
* Added wheel-style picker for:

  * Relay retention
  * Disappearing messages

### Fixes & Improvements

* Fixed DM sync issue:

  * Prevented imported/opened conversations from overwriting local history
* Improved message rebuild logic:

  * Control events (e.g. reactions) are folded into target messages
* Added first-pass live attachment transfer for secure chats
