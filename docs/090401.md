# 📅 LibreChat — Change Log (090401)
First ChangeLogs
For 0.1.1 Aplha

## 🔐 DMs, Privacy & Encryption

* Prevented unknown DM conversations from auto-importing from live delivery/relay.
* Set default DM relay retention to **0** (no server-side backlog by default).
* Ensured DMs cannot be sent before encryption is initialized.
* Added **encryption onboarding flow** with a clear lock-stage UI.
* Fixed encryption bugs:

  * Lock UI no longer disappears mid-animation.
  * Encrypting a chat no longer triggers “forget old conversation”.
* Improved handling when:

  * Conversation exists but device has no local key → starts fresh instead of crashing.
  * Conversations are restarted or unavailable → now shown clearly in UI.
* Added key wrapping for **user’s other devices** when creating fresh DMs.

## 🧠 Conversation Control

* Added **“Forget old conversation” (device-local)** feature.
* Added undo/reset option in conversation settings.
* Fixed freeze caused by this feature (memoized selected friend state).

## 💬 Friends & Messaging UX

* Added **group DMs** on the real DM stack.
* Friend list now shows **last message preview** instead of “DM ready”.
* Added preview truncation (`hey hows it g...`).
* Added:

  * Desktop notifications (DMs + group chats)
  * Unread/highlight glow system (real tracking, not time-based)
  * Theme-aware unread indicators
* Unread clears only when:

  * Chat is open + focused + visible
* Improved layout:

  * Main Friends screen no longer scrolls (only panels do)
* Moved:

  * Add Friend → popup
  * Friend requests → popup
* Added:

  * Collapsible friend tag folders
  * Persistent open/closed state
* Fixed:

  * Context menus (viewport clamping + cursor offset)
  * Duplicate/misplaced add-friend button

## 🏷️ Friend Tags System

* Rebuilt friend tags into **editable folders in Client Settings**
* Grouped:

  * Friends in sidebar by tag
